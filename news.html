Accept Payment API

1. Integrate following link at the payment page.
 <a href="https://portal.azewallet.ulis.co.uk/Dashboard/Pay/Gateway?mid=<MERCHANT_ID>&
 api_key=<API_KEY>&amount=100&cy=KWD&order_id=<ORDER_ID>">Pay Now</a>
2. The user will be redirected to payment page with the AZ Wallet option.
3. Check order status
